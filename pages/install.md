### Manual installation

See [Installation from source](https://github.com/plainDE/docs/wiki/Installation-from-source) wiki page

## Run plainPanel

Now you can add plainPanel (+ xcompmgr, if opacity is needed) to your OpenBox/FluxBox/... autostart and enjoy!

```sh
plainPanel
```
